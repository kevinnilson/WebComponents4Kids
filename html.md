
# HTML

HTML marks the content up into different structural types, like paragraphs, lists, images, tables, etc.



```
<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>This is a Heading</h1>
<p>This is a paragraph.</p>

</body>
</html>
```
